🔗 Unnamed Pipe :

Unnamed Pipes provide communication between related processes (parent-child).

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🧠 Concept :
Created using pipe()
Exists only in memory (not stored)
Supports half-duplex communication

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

⚙️ System Calls Used :
pipe() → Create pipe
fork() → Create child process
read() / write() → Communication

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🔄 Workflow :
Create pipe
Fork process
Parent writes / Child reads (or vice versa)
Data removed after reading

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🌍 Use Cases :
Parent-child communication
Command chaining in shells

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

⚠️ Notes :
Only works between related processes
Data is deleted after reading
No persistence

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
